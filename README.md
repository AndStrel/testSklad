# TemplateRepo

Шаблонный репозиторий для проектов с использованием Vite, TypeScript, SCSS, Husky Jest и Cypress.

## Стек технологий:

- Vite
- TypeScript
- SCSS
- Jest
- Cypress

## Как использовать

1. Склонируйте репозиторий.
2. Установите зависимости: `npm install`.
3. Запустите проект: `npm run dev`.
